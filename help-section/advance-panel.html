Create Campaign



Create campaign by clicking on Create Campaign button and fill in the basic details.

Step 1: Overview
This is super simple. We’ve entered few basic details for you. You can edit the name of your campaign, set the total budget, change the CPC value and select a date range. If you want to cap the per day spends you can also set up a daily budget.



Step 2: Product
Select products for advertising from your already listed products on Snapdeal. You can search your products by SUPC or product name.



Step 3: Location
This is optional step. By default your campaign is served across all India. Incase you want to limit your campaign to select locations as you may not be selling everywhere, this step helps you select where all your ads would appear.



Step 4: Target Audience
This is again an optional step. System has inbuilt intelligence to decide where all to serve ads. If you want to select additional categories for targeting, select the categories here. Don’t worry about inconsistent targeting. System has a check to ensure that the ads are served only on relevant categories. If all the minimal fields are filled in, clicking on Done activates the campaign. Your ads would start delivering from start date.
